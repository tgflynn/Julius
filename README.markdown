# Julius


<a id="orgf2af379"></a>

## Description

This is a very preliminary attempt to create a new Lisp dialect
implemented using Common Lisp.


<a id="orgaa5b98f"></a>

## Dependencies

SBCL and quicklisp

<a id="orgab5b98f"></a>

## Usage

  1. Clone Repository.
  2. cd into julius directory.
  3. Start sbcl as follows: sbcl --load startup.lisp.
